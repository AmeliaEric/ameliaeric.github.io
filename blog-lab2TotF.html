A. Tool Exploration: Xcode Information

Question Information/Answer
Platform compatibility (iOS, Android, Windows Phone) iOS
Application development type (Native, Web, Cross-platform) Native
Advantages and disadvantages of the tool Advantages:
• Has an IDE
• Supports Swift
• Includes a graphical interface builder.
• Includes testing tools.
• Provides built-in support for continuous integration and deployment.

Disadvantages:
• Only available for developing apps for Apple’s platforms (ex. macOS, iOS, watchOS, tvOS).
• Can be overwhelming due to the wide range of features and tools.
• Both Xcode and the projects are large in file size thus taking up space on your computer.
• Can be slow sometimes due to the large file sizes.
The tool's website and developer information Website: https://developer.apple.com/xcode/
Developer Information: Release date: October 23, 2003
Licensing Details: Pricing: Free to Download
Programming languages used Swift, C, C++, Objective-C, Objective-C++, Java, AppleScript, Python, Ruby, ResEdit (Res)

Built-in emulator is available?: No because emulator is designed for features with Android app production but it does have a simulator which mimics an iOS apps production environment.

List any required software installations on the host machine. Making sure your machine has the minimum software required and then you should be able to go to the app store and download Xcode for free.

B: Tool Review

I strongly believe that Xcode has a strong potential for app development. It has a very user friendly IDE which gives many great coding suggestions and clear error messages. Xcode is also very smooth with its simulator and provides a great testing experience for developers, allowing them to test not only their app but go to the home page as well and see how the app cover would look. I used Xcode over the past summer to learn Swift and I can confidently say it is one of the best IDE's I have used, it's capabilities for developers is very large and its usability is solid, I very rarely had any problems and if I did then they were user errors (whoops!). Overall, Xcode is amazing for developing apps as it not only provides a very realistic testing simulation on any Apple device chosen, but is also amazing for code as it provided great suggestions and error messages.
